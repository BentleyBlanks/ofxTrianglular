ofxTrianglular
==============

能够非常简易的将任意2d图形进行三角形细分 底层库为triangle与OpenFrameworks<br><br>
支持到跨平台```OSX```与```Windows```平台(Xcode 6.0, Visual Studio 2012) ```Linux```平台并未测试

####极端情况
在个别的少数极端情况Windows端会比OSX端更容易崩溃 但也仅仅只是在图形极端复杂嵌套的情况下
